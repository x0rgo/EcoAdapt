Pair your bridge

Send WiFi credentials over Bluetooth — no wires, no captive portal.
⚠️ Web Bluetooth requires Chrome, Edge, or Opera on desktop, or Chrome on Android. iOS Safari does not support it — use the captive portal method on iPhone.
⚠️ You're not logged in. Pairing without an API key means your bridge won't be able to send sensor data to your dashboard. Log in first, then come back to this page.

How it works

  1. Power on your bridge (USB).
  2. If it has no WiFi configured, it advertises over BLE for 60 seconds. Look for a device named EcoAdapt-XXXX.
  3. Click Pair below, pick the device.
  4. Enter your WiFi info — bridge connects and reboots.

WiFi networks

Prefer to pre-stage WiFi when flashing? Use the web flasher instead — it bakes WiFi into the firmware so the bridge connects on first boot with no Bluetooth pairing needed.
